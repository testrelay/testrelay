# TestRelay Site

The TestRelay gatsby site serving the front page and docs. 

## Quick start

Install the deps:

```shell
npm install
```

Run the development site which will watch for changes and recompile gatsby:

```shell
npm run develop
```

This will open your browser to a site running at `localhost:8000`
